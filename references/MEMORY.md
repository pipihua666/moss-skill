# MEMORY

`MEMORY.md` 只做人物索引，不存放人物详情。

## Rules

- Search by `name` and `aliases`.
- Keep one index row per person.
- Update the row when adding, renaming, or enriching `references/<名称>.md`.
- Preserve the user's exact wording for aliases when possible.
- Mark uncertain values as `待确认`.
- Keep this file short, structured, and easy to scan.

## 人物索引

| name | aliases | file | relationship | status | last_updated |
| --- | --- | --- | --- | --- | --- |

## 新增人物模板

新增人物时：

1. 在本表新增一行
2. 创建对应文件 `references/<名称>.md`
3. 详细信息只写进人物文件，不写进本索引
